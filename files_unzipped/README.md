# RAHGH — Relation-Aware Heterogeneous Graph Homogenization

## Quick start (3 commands)

```bash
# 1. Install environment (auto-detects CUDA)
bash scripts/install_env.sh

# 2. Download all datasets
python scripts/download_datasets.py

# 3. Verify downloads
python scripts/verify_downloads.py
```

## Datasets

| Dataset    | Nodes         | Relations | Task              | Source    |
|------------|--------------|-----------|-------------------|-----------|
| DBLP       | 26,128 (4 types) | 3      | Author 4-class    | PyG / HGB |
| ACM        | 11,246 (3 types) | 3      | Paper 3-class     | PyG / HGB |
| IMDB       | 11,616 (3 types) | 2      | Movie 3-class     | PyG / HGB |
| OGBN-MAG   | 1.9M (4 types)   | 4      | Paper 349-class   | OGB       |
| Amazon     | ~49K (3 types)   | 3      | Item class / link | PyG / HGB |
| LastFM     | ~27K (3 types)   | 3      | Link prediction   | HGB       |
| Yelp       | variable         | 4+     | Business class    | HGB       |
| Freebase   | variable         | 36+    | Entity class      | HGB       |

## Download options

```bash
# Download everything
python scripts/download_datasets.py

# Download only the 3 standard benchmarks (fastest, ~50 MB)
python scripts/download_datasets.py --only dblp acm imdb

# Skip large / optional datasets
python scripts/download_datasets.py --skip ogbn_mag yelp freebase

# Skip social datasets entirely
python scripts/download_datasets.py --skip amazon lastfm yelp freebase
```

Each dataset gets a `.done` marker file when complete.  
Re-running skips already-downloaded datasets automatically.
